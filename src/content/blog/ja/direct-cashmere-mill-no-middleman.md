# カシミヤ工場直販、中間業者なし：取引会社のマークアップを避けて直接仕入れる方法

# カシミヤ工場直販、中間業者なし：B2Bガイド

カシミヤ業界には **30～40％の中間マージン** が存在します。欧州や北米のほとんどのB2Bバイヤーは、中国のカシミヤ工場から直接購入するのではなく、**香港、イスタンブール、ミラノの商社** から購入しており、そこでは30～40％のマージンが上乗せされ、現地語サポートが提供されています。

1回の出荷で500kg以上を発注するB2Bバイヤーにとって、**工場から直接調達**することで中間業者を排除し、単価を25～35％削減できます。

しかし、直接調達にはトレードオフがあります。このガイドでは、直接調達が有効なケースと、その正しい方法を解説します。

## カシミヤにおける3つの中間業者モデル

### モデル1：香港の商社

EU/米国向けB2B輸出で最も一般的。香港拠点の商社：
- **工場価格に30～40％のマージン**を上乗せ
- **提供するもの**：英語/欧州言語サポート、小口注文（MOQ）の柔軟性、在庫ファイナンス、物流管理
- **最適なケース**：1回の出荷で200kg未満のバイヤー、またはコストよりも利便性を重視する場合

### モデル2：イタリア/スコットランドのコミッションエージェント

超高級品バイヤーに一般的。現地エージェント：
- **20～30％のコミッション**を上乗せするが、現地の品質管理と保管も提供
- **専門分野**：特定の工場（例：2～3の工場のみを代理）
- **最適なケース**：カシミヤ取引において「現地パートナー」を求めるバイヤー

### モデル3：トルコの加工業者

完成品を求めるバイヤー向け。トルコの工場は中国の糸を購入し、現地で編み立て・仕上げを行い、再輸出することが多い。50～80％のマージン（糸＋人件費＋諸経費）を上乗せするが、EU/米国への納期が短い。

**最適なケース**：関税やマーケティング上の理由で「メイド・イン・トルコ」ラベルを求めるバイヤー

## 直接調達が有効なケース

工場直接調達が効果を発揮するのは以下の場合です：

1. **1回の出荷で500kg以上の注文量** — 固定費（出張費、通信費、書類作成費）が単位数に分散される
2. **物流パートナーがいる** — 仕向国にフォワーダー、通関業者、または倉庫がある
3. **英語でコミュニケーションが取れる**（ほとんどの中国工場には英語対応の営業スタッフがいる）
4. **前金30％（T/T）、出荷前に残金70％の支払いが可能**（中国の標準的な支払条件）
5. **30～45日の生産リードタイム＋25～35日の輸送期間を受け入れられる**

1回の出荷で200kg未満の注文、またはこれらの条件を受け入れられない場合は、商社が適切な選択肢です。

## コスト比較：直接調達 vs. 中間業者

同一製品（プレミアムホワイトデヘアードカシミヤ、FOB天津）：

| 調達モデル | 1kgあたりのコスト | 得られるもの |
|---|---|---|
| **工場直販** | $90-105 | 繊維のみ、FOB天津、お客様が配送手配 |
| **香港商社** | $120-145 | 繊維 + 香港倉庫保管 + 英語サポート + お客様の港までのLCL配送 |
| **イタリア代理店** | $130-155 | 繊維 + 現地品質管理 + EU通関手続き + 保管 |

直接調達による「節約額」（25-35%）は確かに実現可能ですが、以下の負担が生じます：
- **出張** — 監査と品質確認のためのオルドスへの訪問（年1-2回、1回あたり$2,000-5,000）
- **コミュニケーションの負担** — 工場は英語対応ですが、時差や休日カレンダーなどをお客様が管理する必要があります
- **物流** — 配送、通関、倉庫保管を自ら手配

1,000kgの注文の場合：
- 直接調達：繊維$90-105K + 物流約$5K = **合計$95-110K**
- 香港商社：配送込み$120-145K = **合計$120-145K**
- **節約額：$25-35K**

節約額は本格的なB2Bバイヤーにとって確かなものです。問題は、直接調達を管理するための業務能力をお持ちかどうかです。

## 直接調達の4つのステップ

### ステップ1：工場への初期アプローチ

- **B2Bプラットフォームを検索**（Alibaba、Made-in-China）「OEKO-TEX カシミヤ工場」 — 正規の工場は認証を表示していることがほとんど
- **展示会に参加**（Première Vision Paris、SpinExpo、Intertextile Shanghai） — 工場と直接面談
- **業界の紹介** — 他のカシミヤバイヤーに工場の推薦を依頼
- **工場が本物か確認**：生産ラインを自社で所有しているか？（ほとんどの商社は所有していない）

**最初の問い合わせを送信**する際に含める内容：
- 対象の繊維/糸/衣料品の種類
- 数量と頻度
- 目標価格帯
- 必要な認証
- 配送先

### ステップ2：工場監査（必須）

**大量注文の前に必ず工場を直接訪問**してください。施設で1-2日過ごすことで、以下を確認できます：

- **実際の生産ライン**（単にオフィスだけの商社ではないこと）
- **社内のOFDA設備**（品質テストを実際に行っていること）
- **牧民から顧客までのベール追跡システム**
- **保管状況**
- **労働力の規模とスキル**

初回の監査には**カシミヤの専門家**を連れて行くか（または現地コンサルタントを雇う、約$500/日）、2-3回の監査後には工場を十分に把握できるため専門家は不要になります。

### ステップ3：トライアル注文

まずは**小ロットのトライアル注文**（通常は200～500kg、または衣料品の場合は100～300着）から始めて、以下をテストします：
- **品質の一貫性** — バルク品はサンプルと一致するか？
- **コミュニケーションのスピード** — 工場の返信にかかる時間は？
- **物流** — 期日通りに出荷できるか？
- **支払い条件** — 合意した支払いスケジュールを守るか？

トライアル注文では、**書類の流れ**もテストできます — 商業送り状、パッキングリスト、原産地証明書、OEKO-TEX認証、OFDAテストレポートなど。

### ステップ4：定期注文へ拡大

トライアルが成功したら、**定期注文**または**ロール予測**モデルに移行します。これにより以下のメリットが得られます：
- **より良い価格設定**（通常、単発注文より5～10％低い）
- **繁忙期（8月～11月）の優先生産**
- **他のバイヤーが争う中での安定供給**

典型的な定期注文：月1,000～2,000kg、3～6ヶ月のロール予測。

## 直接調達における支払いと書類

標準的な中国工場の支払い条件：

- **注文確定時に30％T/T前払い**
- **出荷前に70％T/T残金**
- **新規顧客向けL/C一覧払い**（より安全だが、銀行手数料は200～500ドル）
- **Aグレード顧客向けOA（オープンアカウント）**（取引実績6～12ヶ月後）

工場が提供すべき標準書類：

- **商業送り状**（英語および現地語）
- **パッキングリスト**（重量、寸法、数量）
- **船荷証券**（海上）または**航空貨物運送状**（航空）
- **原産地証明書**（国に応じてForm A、Form E、Form F）
- **OEKO-TEX認証**（該当する場合）
- **OFDAテストレポート**（ベールごと、要求された場合）
- **保険証券**（CIF保険を購入した場合）

**最初の3～5回の注文では、中国の繊維輸出に精通したフォワーダーを必ず利用してください**。通関書類、港湾手続き、出荷調整を1回の出荷につき200～500ドルで代行します。

## 直接調達の5つのリスク（とその管理方法）

### リスク1：品質の不一致

サンプルは素晴らしいが、バルク品は平凡。

**対策**：発注書にベールごとのOFDAテストを明記。仕様を満たさないベールは拒否。ほとんどの信頼できる工場は品質問題による返品を受け入れます。

### リスク2：納期遅延

中国の工場は第3四半期～第4四半期（繁忙期）に生産能力の問題を抱えることが多い。2～4週間の遅延は一般的。

**対策**：バッファ在庫を確保。必要と思うより3～4週間早く注文する。

### リスク3：コミュニケーションの問題

タイムゾーン、言語のニュアンス、祝日スケジュール（春節、国慶節）。

**対策**：書面記録にはメールを使用。毎週の電話会議を設定。期待事項を明確に伝える。

### リスク4：為替変動

人民元-米ドルの為替レートは年間5～10％変動する可能性があります。

**対策**：5万米ドル以上の注文には先物契約を利用。またはL/Cで米ドル建て支払い。

### リスク5：知的財産保護

中国の工場にデザインを送ることは、コピーのリスクを内在的に伴います。

**対策**：NDA（秘密保持契約）を締結。まずは重要でないデザインから始める。2～3年かけて信頼を構築してから、最高の作品を送る。
