# OEKO-TEX認証カシミア：B2Bバイヤーが知るべき繊維安全基準

# OEKO-TEX認証カシミヤ：B2Bバイヤーが知っておくべきこと

カシミヤは天然繊維です。しかし、ヤギから衣料品に至る過程では、**化学物質**（精練剤、染料、仕上げ処理剤、防ピリング酵素など）が使用されます。これらの化学物質の中には、人体に有害な残留物を残すものがあります。

EU、米国、日本に輸出するB2Bバイヤーにとって、繊維製品の安全性認証は、**法的に義務付けられているか、バイヤーから要求される**ケースが増えています。このガイドでは、認証の実際の意味、重要度、および主張を検証する方法について説明します。

## 7つの主要な繊維製品安全性認証

### 1. **OEKO-TEX Standard 100**（最も重要）

**試験内容**: ホルムアルデヒド、重金属（鉛、カドミウム、水銀）、農薬、クロロフェノール、フタル酸エステル、アゾ染料など300以上の物質を検査。

**レベル**:
- **クラスI**（乳幼児用）：最も厳格、ほとんどの物質で閾値0.5%未満
- **クラスII**（肌に直接触れるもの）：一般的な大人用衣料品の基準
- **クラスIII**（肌に直接触れないもの）：ホームテキスタイル
- **クラスIV**（装飾のみ）：ほとんど使用されない

**試験対象外**: 動物福祉、環境影響、繊維の原産地。

**工場のコスト**: 年間2,000～5,000ドル + 出荷ごとの試験費用。

### 2. **GOTS（グローバルオーガニックテキスタイルスタンダード）**

**認証内容**: 有機繊維含有率（「オーガニック」表示には70%以上、「有機」ラベルには95%以上）+ 環境に配慮した加工 + 社会的基準。

**認証対象外**: 動物福祉（カシミヤヤギは、繊維が「有機」であってもGOTSの対象外です。これは既知のギャップです）。

**工場のコスト**: 年間3,000～8,000ドル + サプライチェーン管理文書。

### 3. **RWS（責任あるウール基準）**

**認証内容**: 動物福祉 + 土地管理 + ウール（および最近ではカシミヤ）のサプライチェーン管理。

**認証対象外**: 化学物質の安全性（OEKO-TEXのような物質試験はなし）。

**工場のコスト**: 年間2,500～6,000ドル。

### 4. **ISO 9001**（品質管理）

**認証内容**: 品質マネジメントシステム。製品の安全性ではありません。

**認証対象外**: 実際のカシミヤや使用される化学物質に関する一切の事項。

**工場のコスト**: 初回3,000～15,000ドル + 年次監査。

### 5. **ISO 14001**（環境管理）

**認証内容**: 環境マネジメントシステム。

**認証対象外**: 製品の安全性や繊維の原産地。

### 6. **BSCI / Sedex SMETA**（社会的コンプライアンス）

**認証内容**: 労働者の福祉、公正な賃金、児童労働の禁止。

**認証対象外**: 製品の安全性や繊維の原産地。

### 7. **GCS（グッドカシミヤ基準）**（業界固有）

**認証内容**: カシミヤ繊維含有率 + 動物福祉 + 農場から完成品までのトレーサビリティ。

**認証対象外**: 化学的安全性（物質テストなし）。

## カシミヤにとって実際に重要な認証

異なる市場のB2Bバイヤー向け：

| 市場 | 必須/義務 | 推奨 |
|---|---|---|
| **EU（全市場）** | REACH準拠、OEKO-TEX（ほとんどの小売業者が必須） | GOTS、RWS |
| **米国** | CPSIA準拠、OEKO-TEX（百貨店が必須） | RWS、GOTS |
| **日本** | OEKO-TEX（標準）、JIS L 0217（日本規格） | RWS |
| **中国（国内）** | GB 18401（中国繊維安全基準） | OEKO-TEX |
| **韓国** | KCマーク（OEKO-TEXに類似） | RWS |

**B2Bバイヤーの90%にとって、OEKO-TEX Standard 100（クラスII）が最低基準です。** それ以下では、ほとんどのEU/米国の百貨店や高級小売業者に販売できません。

## 認証主張の確認方法

どの工場でも認証を主張できます。確認方法は以下の通りです：

### ステップ1：証明書番号を尋ねる

すべてのOEKO-TEX証明書には固有の番号があります。サプライヤーに証明書番号と認証機関（例：「Hohenstein Institute」または「Testex」）を尋ねてください。

### ステップ2：認証機関のウェブサイトで確認する

- **OEKO-TEX**: https://www.oeko-tex.com/en/buyers/search（会社名または証明書番号で検索）
- **GOTS**: https://www.global-standard.org/find-suppliers.html
- **RWS**: https://textileexchange.org/certificate-search/
- **BSCI**: https://www.bsciplatform.com/

証明書が表示されない場合、その主張は虚偽です。

### ステップ3：証明書の範囲を確認する

証明書は特定の製品カテゴリを対象としています。工場のOEKO-TEX証明書が「糸と生地」を対象としていても「衣料品」を対象としていない場合、完成したニットウェアは実際には認証されていません。

### ステップ4：有効期限を確認する

OEKO-TEX証明書の有効期間は通常1年です。「6ヶ月前に期限切れ」の証明書は技術的に無効です。

## サプライヤーが認証を偽造した場合の対処法

これは想像以上に一般的です。2023年の業界調査では、中国のサプライヤーによる「OEKO-TEX認証済み」の主張の15～20%が検証不可能と推定されました。

不正の疑いがある場合：
1. **証明書のPDF**（番号のみでなく）を要求し、そのリンク先URLを確認してください
2. **証明書番号**を認証機関のウェブサイトで検証してください
3. **サンプル出荷品の独自テスト**を依頼してください（1テストあたり200～500米ドル）
4. 不正が確認された場合は**認証機関に報告**してください — サプライヤーの認定を取り消すことができます

## 一般的なOEKO-TEX証明書の見方

標準的な証明書には以下が含まれます：
- **証明書番号**（例：「SH015 123456」）
- **発行日**と**有効期限**（通常1年間）
- **保有者**（認証を受けた企業）
- **製品クラス**（I、II、III、またはIV）
- **試験対象品目**（糸、生地、衣料品など）
- **試験機関**（Hohenstein、Testexなど）
- **権限のある署名**と印鑑

これらのいずれかが欠けている、または改ざんされているように見える場合は、認証機関から直接新しいコピーを請求してください。
